How It's Made 

Carpe Diem was made with the following technologies:

  • JavaScript Libraries:
    • jQuery
    • BootStrap
    • Moment
  • API Calls:
    • NASA - Image of the Day
    • Ninja API - Quote of the Day
    • OpenWeather - All Weather Info
    • Google - Calendar & Tasks API
    • MapQuest - Local Traffic
    • Cats API - Need I say more?
  • Designed with BootStrap Studio and Developed with VS Code
  • Hosted on GitHub Pages
