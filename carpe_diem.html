How It's Made

Carpe Diem was made with the following technologies:

  • Modern JavaScript ES6+
  • Modern CSS with custom properties and flexbox/grid layouts
  • API Calls:
    • NASA - Image of the Day
    • Ninja API - Quote of the Day
    • OpenWeather - All Weather Info
    • Google - Calendar & Tasks API
    • MapQuest - Local Traffic
    • Cats API - Need I say more?
  • Features:
    • Responsive design that works on all devices
    • Dark/light mode toggle
    • Pomodoro timer for productivity
    • Smooth animations and transitions
    • Modern UI with cards and clean typography
    • Accessibility improvements
