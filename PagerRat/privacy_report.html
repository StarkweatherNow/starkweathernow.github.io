2. False Positives & Warnings Report

Static analysis tools often flag standard system libraries and features as "Warnings." This table analyzes the warnings found in the report and explains why they are safe.

Finding Reason Flagged Actual Status
Cleartext Traffic Permitted
Network 		Domain config is insecurely configured to permit clear text traffic to 172.16.52.1. Native Feature
This configuration allows the application to mimic the native Remote Pager Access functionality built into the device's original firmware.
Permission: BIND_REMOTEVIEWS
Manifest 		Service is exported to other apps. Required Feature
Necessary for the Home Screen Widget to function. Protected by system-level permissions.
MD5 is a weak hash
Cryptography 		Scanner found `MD5Digest` class in Bouncy Castle library. False Positive
Artifact of the `bcprov` library. The app explicitly enforces AES-256-CTR and does not use MD5 for encryption.
SHA-1 is a weak hash
Cryptography 		Scanner found `SHA1Digest` class in Bouncy Castle library. False Positive
Artifact of the `bcprov` library. The app config disables weak ciphers. The class exists in the binary but is never invoked.
Services: androidx.work
Library 		Background services have broad permissions. Google Library
This is WorkManager, Google's standard library for safe background tasks (e.g., SSH keep-alives).
Receiver: ProfileInstaller
Library 		Exported broadcast receiver found. Google Library
Standard Google library that optimizes app installation speed via the Play Store.
Execute Raw SQL Query
Database 		Scanner found SQL keywords in code. False Positive
We use Android Room. It generates secure SQL at compile-time and uses parameterized queries, preventing injection attacks.
Insecure Random Number Generator
Code Quality 		Usage of `java.util.UUID` detected. Acceptable Risk
Used solely for generating database primary keys (UUIDs). Cryptographic keys are generated using Bouncy Castle's SecureRandom.
IP Address / Hardcoded User
Security 		Default string literals found in `AddDeviceViewModel`. Test Data
These are UI placeholders (e.g., "172.16.52.1", "root") for the "Add Device" form, not active credentials.
App copies data to clipboard
Code 		Code exists to write to clipboard. Intended Feature
The app includes the ability to copy terminal output or command snippets to the clipboard for user convenience.
App can read/write to External Storage
Code 		Daemon scanner detects external storage devices. False Positive
No Android Permission Request: The Android application does not request READ_EXTERNAL_STORAGE or WRITE_EXTERNAL_STORAGE permissions from the Android OS.
Trusts System Certificates
Network 		Config explicitly mentions system CAs. Standard Behavior
The app trusts standard Android System CAs (like Google/Verisign) but blocks user-installed proxy certs to prevent MITM.

How to verify on Android:

  1. Install App Manager from F-Droid (or a similar package inspector).
  2. Open App Manager and search for Pager Rat.
  3. Scroll down to the Signatures section.
  4. Compare the SHA-256 string on your screen with the key above. If they match, the app is authentic.
